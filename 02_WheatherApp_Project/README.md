# 🌤️ Modern Weather Dashboard

A beautiful, responsive, and interactive weather application built with a modern glassmorphism UI. Get real-time weather updates, a 5-day forecast, and seamlessly manage your recent searches.

## ✨ Features
* **Real-Time Weather Data:** Accurate current conditions, temperature, humidity, wind speed, and pressure.
* **5-Day Forecast:** Reliable 5-day outlook.
* **Geolocation Support:** Automatic weather detection for your current location.
* **Search History:** Save recent searches locally.
* **Dynamic UI:** Glassmorphism design with smooth animations and responsive layouts.
* **Unit Toggling:** Switch between Celsius (°C) and Fahrenheit (°F).
* **Dark/Light Mode:** Theme toggling for optimal viewing.

## 🛠️ Technologies Used
* **HTML5:** Semantic markup.
* **CSS3:** Glassmorphism (`backdrop-filter`), Flexbox, Grid, and Animations.
* **JavaScript (ES6+):** Fetch API, DOM manipulation, and LocalStorage.

## 🚀 Getting Started

### Prerequisites
You will need a free API key from [OpenWeatherMap](https://openweathermap.org/api).

### Installation
1. **Clone the repo:** `git clone https://github.com/your-username/weather-dashboard.git`
2. **Navigate to the folder:** `cd weather-dashboard`
3. **Add your API Key:** Open `pr.js` and replace the placeholder with your actual key.
4. **Run the app:** Open `index.html` in your browser or use a "Live Server" extension.

## 📂 Project Structure
* `index.html`: Main structure.
* `style.css`: Styling and glassmorphism effects.
* `pr.js`: API logic and data handling.
* `README.md`: Project documentation.
